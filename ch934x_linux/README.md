1. BUILDING
 $ sudo make

2. LOAD
 $ sudo make load
 or you can use
 $ sudo insmod ch934x.ko
 
3. UNLOAD
 $ sudo make unload
 or you can use
 $ sudo rmmod ch934x.ko
 
4. AUTOLOAD SINCE BOOT
  $ sudo make install
  
5. CANCEL AUTOLOAD SINCE BOOT
  $ sudo make uninstall
